# AI-Powered FAQ Chatbot

An intelligent FAQ chatbot developed using HTML, CSS, and JavaScript that provides instant answers to frequently asked questions using Natural Language Processing techniques.

The chatbot utilizes TF-IDF vectorization and Cosine Similarity to identify the most relevant response from a predefined knowledge base. It features a modern chat interface, multiple FAQ categories, confidence scoring, suggested questions, and real-time interactions.

## Features

* Interactive chat interface
* NLP-based question matching
* TF-IDF text vectorization
* Cosine Similarity search
* Multiple FAQ categories
* Confidence score display
* Suggested follow-up questions
* Responsive design
* Real-time chatbot responses

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Natural Language Processing (NLP)
* TF-IDF Algorithm
* Cosine Similarity
